Ferramentas Utilizadas

Para se desenvolver uma página web é recomendável a utilização do programa Visual Studio Code, pois é um software open-source, multi-plataforma, grátis e oferece ao usuário diversas extensões e plugins para facilitar na hora de por em prática o desenvolvimento.

Para fazer o download basta acessar este link e selecionar a versão desejada. Após isso, é so seguir o processo de instalação, selecionando a pasta de instalação e concluir a instalação.

O recomendável a se fazer ao iniciar pela primeira vez para desenvolvimento web, é instalar extensões para aumentar a produtivdade, para isso basta ir no menu do canto esquerdo, clicar em extensões, e procurar a extensão desejada, no caso a extensão Live Server, para abrir um servidor local para abrir seus arquivos HTML com recarregamento automática da página. Outra extensão mas que já vem pré-instalada que será util no desenvolvimento, é a Emmet. Nela terão comandos que facilitarão a escrita de seus códigos.

Outra ferramenta que ajudará ainda mais na produtividade, é o inspetor de elementos do navegador. Caso esteja utilizando o Google Chrome basta apertar F12. Estrutura básica do HTML

(Voltar ao índice)

A estrutura básica do HTML, consiste na utilização por tags, que são os elementos que formam uma página HTML. Nela ocorre a separação por <html>, <head> e <body>, a tag <html>, é utilizada pra demarcar o espaço em que vai ser utilizado para fazer códigos em HTML. na head ou cabeça de uma página HTML, é onde é carregado configurações antes de exibir para o usuário, por exemplo, pode ser definido um título pela tag <title>. E no body ou corpo, é onde é mostrado para o usuário todas as suas tags, formando uma página HTML por títulos, parágrafos, imagens etc.

Para ver o seu arquivo HTML no navegador, basta clicar na opção "Go Live", no menu inferior do Visual Studio Code com a extensão Live Server instalada.

Falando sobre Tags

(Voltar ao índice)

No HTML existem milhares de tags, cada uma com sua função, uma delas é a tag <i>, que serve para transformar o texto que está entre ela em itálico, ou a tag <strong>, que faz com que o texto fique em negrito..

também é possível utilizar tags dentro de outras tags como, nesse exemplo, fazer com que o texto fique em negrito e em itálico.

Outra tag muito importante é o input, que faz com que abra um espaço para que o usuário digite algum dado. e nele existe o atributo type, que diz respeito ao tipo de entrada de dado, se seria um número, texto, senha, email etc.

Caso queira colocar uma imagem existe a tag img, em que no atributo src poderá ser específicado o link, ou o diretório para aquela imagem, juntamente com o width, em que poderá ser ajustado o tamanho da imagem.

Atributos das tags

(Voltar ao índice)

No HTML, existem os atributos das tags, que são propriedas que uma tag deve ter. Existem atributos globais que são os que quase toda tag tem e os específicos que somente algumas tags tem.

Nas tags em geral, pode ser utilizado o atributo "id", que faz referência ao identificador daquele elemento, podendo ser usado para realizar alguma função no JavaScript, tornando a página mais flexível, ou a class, utilizada no CSS para estilizar algum elemento.

A tag input tem o atributo type, que como ja foi dito anteriormente, muda a entrada de dados do usuário ou o atributo src da tag img, que diz respeito ao local em que essa imagem está inserida para ser mostrada na página HTML. esse é um exemplo de específico.

Textos

(Voltar ao índice)

Existem tags específicas para diversas funcionalidades, aqui estarão tags para serem usadas em textos.

Para colocar títulos na sua página HTML, utilize o h1, h2, h3,h4 até o h6. Esses títulos tem uma ordem de hierarquia, de título e subtítulo, descrição etc. Como se fosse uma página de jornal em que tem o título principal que seria o h1, o subtítulo que seria o h2, e assim por diante.

Também existe o parágrafo, que é a tag <p>, que é usado no corpo do texto de sua página HTML, para escrever textos em geral.

A tag <blockquote>,

faz com que dê um espaçamento e dizendo ao navegador que ele é uma citação.

Além desses, existe a tag <u>, que é o sublinhado.

Caso queira marcar algum texto utilize a tag <mark> que faz com que um texto fique marcado.

Listas ordenadas e Não Ordenadas

(Voltar ao índice)

No HTMl, existem dois tipos de listas, as ordenadas e as não ordenadas. Elas utilizam as tags <ol> de listas ordenadas e <ul> de listas não ordenadas, e cada item dentro da lista utiliza a tag <li>.

Exemplo de listas ordenadas, que tem uma ordem:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplos de listas não ordenadas, que não seguem uma ordem:

  • Item 1
  • Item 2
  • Item 3

Também é possível colocar listas dentro de outras listas.

  1. Item 1
      2. Item 1.1
      Item 1.2
      Item 1.3
  2. Item 2
  3. Item 3

Links

(Voltar ao índice)

Para fazer a navegação entre links externos e para próprias páginas existe a tag <a>, de âncora.

Nessa tag tem um atributo chamado href que especifica aonde que vai ser levado aquela âncora.

No atributo href é possível colocar uma URL, ou o diretório de outro arquivo HTML.

Existe outro atributo chamado target, que trata-se da ação ao clicar no link, se vai ser levado para outra página, redirecionado da na mesma página. Caso queira que seja levado a uma nova aba, "_blank". Caso queira que seja redirecionado da mesma página, que é o comportamento padrão da tag a, "__self"
